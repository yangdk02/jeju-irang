# 제주아이랑 초간단 검수자 매뉴얼

## 평소에는 두 단계만 합니다

### 1. 검수할 행 선택

Google Sheet의 `review_queue`에서 확인할 요청 행의 아무 셀이나 클릭합니다.

### 2. 승인·반영 버튼 누르기

`🍊 제주아이랑 검수 → 선택 행 승인·반영`을 누릅니다.

확인창에 다음 내용이 한국어로 정리되어 나타납니다.

- 장소명
- 신규 등록 정보 또는 변경 예정 정보
- 위치 확인이 필요하면 VWorld 장소명과 주소

내용이 맞으면 `예`를 누릅니다.

이것으로 끝입니다. 다음 작업은 자동으로 처리됩니다.

- 요청 승인
- VWorld 위치 확정
- 신규 장소 추가 또는 기존 장소 수정
- 백업 CSV 생성
- `jeju_irang_master`와 `jeju_irang_export` 갱신
- 최종 CSV 생성
- 처리 로그 기록

성공하면 `완료` 창이 나타나고 해당 행은 `APPLIED`로 바뀝니다.

## 확인창의 내용이 틀릴 때만

`아니오`를 누르고 해당 행의 잘못된 값만 수정합니다.

- 장소 정보가 틀림: 해당 `approved_*` 셀 수정
- 수정할 항목이 틀림: `apply_fields` 수정
- 기존 값을 삭제해야 함: `clear_fields`에 컬럼명 입력
- VWorld 장소가 틀림: 올바른 `source_*` 후보 확인

수정한 다음 같은 행을 선택하고 `선택 행 승인·반영`을 다시 누릅니다.

## 승인하면 안 되는 경우

다음 요청은 버튼을 누르지 않습니다.

- 기존 장소와 중복됨
- VWorld 장소명이나 주소가 다른 장소임
- 사용자 입력만으로 장소를 특정할 수 없음

필요하면 `admin_note`에 이유만 적어 둡니다.

## 처음 한 번만 설정

1. 응답 Google Spreadsheet에서 `확장 프로그램 → Apps Script`를 엽니다.
2. 최신 `csv_sync_export.gs` 코드를 붙여 넣고 저장합니다.
3. Spreadsheet를 새로고침합니다.
4. 상단에 `🍊 제주아이랑 검수` 메뉴가 나타나는지 확인합니다.

시트 안에 큰 버튼을 만들고 싶다면 버튼에 다음 함수명을 할당합니다.

```text
approveAndSyncSelectedRequest
```
